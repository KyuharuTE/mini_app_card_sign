# 简介

这是一个项目通过 NapCat 机器人获得签名后的小程序卡卡片

- 在 2025 年 10 月左右某大厂对各种 Ark 的图片连接添加了代理, 导致 iP 探针无法使用, 本项目绕过了这一点, 本项目生成的小程序卡卡片的图片连接依然是图片原连接

# 如何使用

1. 部署一个 `NapCatQQ` 并开放一个 `WebSocket` 服务端
2. 前往部署好的站点使用, 例如: [https://afrit.cn/miniark](https://afrit.cn/miniark)

# 如何部署

1. 安装 `NodeJs` 和 `pnpm`
2. 在项目文件夹下执行 `pnpm build`
3. 将构建好的网页部署在网页服务器上
